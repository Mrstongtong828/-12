## 二、原创性承诺

本参赛队郑重声明:本作品所提交的全部程序源代码、技术文档、研究报告、操作手册、演示视频及相关附件,系本参赛队在指导教师指导下独立研发,未抄袭、未剽窃、未复制任何已发表作品或网络公开方案的核心实现逻辑。作品中涉及的关键技术方案均由本团队原创设计,具体包括但不限于:面向二十种敏感类型的正则模式库编排、基于百家姓首字锚点与黑名单扩展的中文姓名识别体系、PaddleOCR 后处理字符纠错映射图、数字混淆场景下的 BFS 候选串枚举策略、递归解码链的四重熔断与深度控制机制、分级分类与动态脱敏切面的低耦合织入方案、基于 RBAC 的权限管控策略与审计日志联动模型、以及面向结构化/半结构化/编码/非结构化文本/二进制 BLOB/数据库对象六种形态的统一扫描流水线。本作品所引用的开源组件均严格遵守其对应开源许可协议,未违反任何条款,未使用任何商业付费组件的未授权副本。如本作品后续被查实存在抄袭、剽窃、未经授权使用他人知识产权或其他学术不端行为,由本参赛队伍承担全部法律责任与相应后果,并同意主办方对获奖资格予以撤销。

## 四、数据与模型合规

- 扫描对象:赛题方在指定 Docker Compose 环境中预置的 4 个离线数据库(两个 MySQL 实例、两个 PostgreSQL 实例),所有数据均为赛题方构造的仿真样本,不含任何真实自然人的个人信息。
- 模型权重:本作品仅使用 PaddleOCR 官方发布的 ch_PP-OCRv4 检测模型、识别模型以及方向分类模型三个权重文件,均随交付包本地打包提交,未经二次训练或权重篡改。
- 断网约束:作品运行全程不发起任何外部网络请求,OCR 初始化参数显式关闭自动下载,已通过 `netstat` 抽检与防火墙断网环境下的回归测试,确认零出站连接。
